Dota 2 Update - May 3rd, 2016

  • Fixed some scenarios where the base mana capacity could be lower than intended

  • Added the ability to lock items in your inventory to prevent them from combining with other items

Dota 2 Update - May 5th, 2016

  • Fixed some cases where users could crash the dedicated server instance. Earlier this week we deployed better reporting of server crashes and using this better data we've already been able to fix three different exploits that a small number of players had been using. The responsible players will be VAC banned and we will continue to closely monitor for server exploits.
